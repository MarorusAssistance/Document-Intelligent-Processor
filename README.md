# Document-Intelligent-Processor
Servicio para procesar documentos físicos de distintos formatos y clases para insertar sus datos o información en el almacenamiento en un almacenamiento digital, para luego poder trabajar con estos datos con IA: Para consultar datos entre documentos, sacar métricas para mejorar ventas, sacar informes mucho más comodamente y mucho más desarrollados.
